# vue-button

> Button component for Vue.js

## Install / Usage

NPM Based WorkFlows

``` bash
$ npm install vue-button
```

```html
<template>
  <div id="my-app">
    <v-button>
      Hello
    </v-button>
  </div>
</template>

<script>
import VueButton from 'vue-button'
export default {
  components: {
    VueButton
  }
}
</script>
```
